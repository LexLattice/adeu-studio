# External Prototype Intake

Status: repo-owned intake area for imported external prototype bundles.

Authority layer: support / maintainer intake note only.

This directory is for substantial prototype imports that are grounded in the repo but
are not yet accepted as repo-native internal ADEU work. These packs are preserved here
so they can be inspected, normalized, and later re-authored into the repo's own
planning, lock, verification, and closeout sequence.

Intake doctrine:

- treat imported prototype bundles as the `external single-PR lane` from
  [docs/DRAFT_EXTERNAL_CONTRIBUTION_ALIGNMENT_v0.md](/home/rose/work/LexLattice/odeu/docs/DRAFT_EXTERNAL_CONTRIBUTION_ALIGNMENT_v0.md);
- preserve `claimed_scope`, `observed_reachable_surfaces`, and `accepted_shipped_scope`
  separately;
- default precedent status to `non_precedent`;
- do not unzip imported overlays directly into live app/package paths before a
  maintainer-owned integration decision exists;
- keep narrative notes and source payloads together inside each intake pack.
