seed-intent / charter / residual / loop-state / continuation-decision lineage and

duration, communication law, continuity law, replay / resume law, repo authority,
execute authority, dispatch authority, product authority, or hidden-cognition
governance.

register/checker/test path over the existing `adeu_agentic_de` package and thin
script seam, but not live behavior mutation, checkpoint-law mutation, ticket-law
mutation, charter mutation, residual mutation, continuation-refresh mutation,
communication packet law, office binding, repo-authority widening, replay / resume
widening, execute widening, delegated / external dispatch, product rollout as
authority, or hidden-cognition governance.

not change starter seed ingress semantics, task-charter semantics, task-residual
semantics, loop-state semantics, continuation-refresh semantics, checkpoint
semantics, ticket semantics, continuity semantics, communication semantics,
selected-path semantics, diagnostics semantics, or exit behavior by default.
