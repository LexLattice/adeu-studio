## Purpose

Freeze the bounded ANM-native documentation practice starter for `V66-A` so the
repo can add one typed ANM source-set manifest, one typed ANM document authority
profile surface, and one typed ANM document-class policy surface over the already
released `V47` ANM substrate without reopening `D@1` language semantics,
ownership-transition doctrine, policy-consumer doctrine, repo-wide markdown
supersession, generated-reader authority, or flat authority collapse between lock,
architecture, planning, support, historical, and non-governance docs.

`vNext+182` authorizes docs plus the first `V66-A` implementation path over
existing repo-owned packages, but not `D@1` language widening, generated reader
projection, semantic diff reports, full migration binding profiles, stable compile
report artifacts, repo-wide `.adeu.md` rename posture, implicit markdown
supersession, runtime authority minting, schema release beyond the bounded starter
artifact set, advisory-to-live promotion, or hidden-cognition governance.

In `vNext+182`, the new `V66-A` outputs are document-governance starter surfaces
only. They do not change current runtime behavior, current markdown authority,
released `V47` ANM source semantics, `V62` through `V65` runtime family behavior,
or generated-reader-view posture by default.

    - `.adeu.md` presence is not supersession by itself
    - companion presence is not supersession by itself

      itself

      itself

    - generated reader views remain deferred and are not authority by implication
  - one explicit ordering law:
    - `V66-A` is downstream of the closed `V47` ANM substrate
    - it does not replace `V47` language / IR / checker / evaluator ownership
    - it does not replace current markdown authority
    - it widens source discovery / document authority / class policy first
    - migration binding remains deferred to `V66-B`
    - reader projections and semantic diffs remain deferred to `V66-B`
    - advisory adoption hardening remains deferred to `V66-C`
  - one explicit posture law:
    - source-set manifest remains typed and replayable
    - document authority profile remains typed and replayable
    - document-class policy remains typed and replayable
    - same discovered doc inventory plus same governed source-set inclusion rules
      plus same frozen policy yields the same starter outputs
    - malformed authority profile fails closed
    - contradictory classification posture fails closed
    - contradictory host / companion linkage fails closed
    - unregistered companions fail closed
    - supersession claims without explicit transition law fail closed
    - unknown governance-bearing classification remains explicit rather than
      defaulting silently
    - plain support or historical markdown may remain outside the governed ANM
      source set unless explicitly opted in
