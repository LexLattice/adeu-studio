## Implication For Future Arc Drafting

The next family plan should not treat recursive candidate intake as only external-paper,
external-repo, or external-tool intake. The harness can create new candidate types by
using itself. Those candidates need the same typed discipline as imported candidates:

- source and lineage;
- O/E/D/U mapping;
- evidence and tool-applicability posture;
- authority layer;
- candidate status;
- ratification or rejection path;
- outcome review.

The main `v2` mapping is therefore stronger than either prior variant alone because it
uses the loop's residue as evidence, while keeping the residue support-bound.
